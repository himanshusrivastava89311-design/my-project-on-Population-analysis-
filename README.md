# my-project-on-Population-analysis-This project analyze population data using python

Overview

This project analyzes the population data using python.
Different types of graphs ar used to vizualize the population growth of India and compares it with the world population.

Tools and liberies used:

Python
Pandas 
Matplotlib

Features

Load dataseet using Pandas
create bar graph
create pie chart
compare india population with world population
Add graph titles and labels
Data visualization using python

Graph used:

Bar graph-Shows population growth by year
Pie chart-Shows India's share in the world population

Conclusion-
From the graphs,we conclude that India has a very large share of the world population.The pie chart shows that India accounts for about 17.7% of the world population,making it the most populated countries in the world.
